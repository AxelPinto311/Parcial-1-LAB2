1. ¿Qué diferencia hay entre un error de sintaxis y un error de lógica (falla)?
Los errores de sintaxis son faciles de corregir y se muestran cuando el programa se compila. y de esa forma se pueden buscar facilmente para corregirlos. Los errores de lógica se dan al realizarr un mal algoritmo, por mas que nuestro codigo funcione, el programa no realiza lo que queremos, para evitar estos errores lo mejor que podemos hacer es realizar un analisis previo para saber bien como empezar a programar.
